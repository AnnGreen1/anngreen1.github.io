# HTML网页页脚的版权标志字符

```html
<div class="pagefoot">
    <p>&copy; 2022 Copyright AnnGreen</p>
    <p>20022/02/26</p>
</div>
```
或者
```html
<footer>
    <p>&copy; 2022 Copyright AnnGreen</p>
    <p>20022/02/26</p>
</footer>
```
都差不多、就是一个自己写的div与footer的区别
