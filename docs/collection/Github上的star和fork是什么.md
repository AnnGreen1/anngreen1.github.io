# Github上的watch、star和fork分别是什么意思呢？

1、watch可以用来设置接收邮件提醒

2、如果想持续关注该项目就star一下

3、如果想将项目拷贝一份到自己的账号下就fork

fork就相当于在原本的项目分支上建立一个分支，这个分支属于你自己，你也可以任意修改。如果想将你修改后的代码整合到原有的项目中，需要做pull request操作，当然这得经过作者同意。
